## Moon Hopper
A flappy-inspired mini game featuring a pixel-art moon dodging cosmic pipes.

### Run locally
You can launch the game in any modern browser.

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

### Controls
- **Start** button to launch a run.
- **Tap / Click / Space** to hop.
- **R** to reset.
- Choose **Difficulty** (Chill, Classic, Cosmic) for different pipe gaps and speed.
